查找可用的鏡像
multipass find

查看已創建的實例
multipass list

添加一個實例(第一次需要聯網下載)
multipass launch -n d1 -c 1 -m 1G -d 10G
-n, --name: 實例名稱
-c, --cpus: CPU核心數, 默認: 1
-m, --mem : 內存大小,  默認: 1G
-d, --disk: 硬盤大小,  默認: 5G

運行實例下的命令
multipass exec d1 -- lsb_release -a
multipass exec d1 -- sudo apt update

進入實例
multipass shell d1

掛載宿主機上的一個目錄 & 卸載共享目錄
multipass mount ~/Documents/Share d1:/home/ubuntu/share
multipass umount d1

停止 & 啟動實例
multipass stop d1 d2
multipass start d1

清理
multipass delete d1
永久刪除已delete掉的實例
multipass purge
